# PvP Leaderboard

Website:
https://pvp-leaderboard.com/

Matchmaking:
Skill based matchmaking can be accessed after 20 kills + deaths by clicking the white skull on the side

How the ratings work:
This is a trueskill based system. You gain more rating if you beat someone higher rated, and gain less rating if you beat someone lower rated.

Tournaments/Events:
Coming Late Summer or Early Fall 2026

IP Warning:
This plugin requires an IP in order to submit who won/lost to the live leaderboard. Any information that traverses the internet requires an IP, just like visiting a website does.
