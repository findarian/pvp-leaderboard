# PvP Leaderboard RuneLite Plugin

##TLDR
The plugin submits who you fought and if you won or lost when someone dies, then your rating is updated on the correct leaderboard. There are NH/Veng/Multi/DMM leaderboard currently.

## Rating / Anti-Smurfing
Your rating is bound to you, if you share accounts or use alts, it doesn't matter. Your rating is yours and is the same across all accounts.
You can turn everything except submitting the matches off. Anything you see displayed can be toggled or adjusted how you like.

## Website vs Plugin Side Panel (White Skull)
You can look up people via the plugin or have it open a website. Both have the same data.
Match history There is a popout menu for match history for any player you look up in the side panel (White skull on the right hand side of runelite)

## Networking, performance & logging
Requests are done to the website or the API. I work in cybersecurity and put my name on this project so you have more assurance that I know how to handle your data and do so safely.

## Authentication
- Not many features currently, you can ignore if you want to!
- Optional Cognito OAuth; tokens are refreshed in the background using the refresh token.
- This gives you additional stats for plugin lookup and on the website
